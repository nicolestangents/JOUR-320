Exploring Flourish

While I worked with Datawrapper during my internship with Midstory, this was my first time exploring Flourish's functions. I decided to examine a dataset on food inspections in Chicago from 2010 to the present, taken from the Chicago Data Portal . I wanted to translate the data to tell a story about establishments that have failed food inspections across the Chicago area.

1. Filtering the data set

I downloaded the raw data from the Chicago Data Portal in .csv form. Because the file was too large to open in Google Sheets, I manipulated it in Excel.

The goal was to tell a story through both a chart and a map.


2. Creating a pivot table

To select the data I wanted for my chart, I created a pivot table (insert > pivot table). Sorting by ZIP codes, I filtered the data to only show a count of the inspection results — specifically, pass and fail (excluding out of business, pass with conditions, not ready, etc.). I manually filtered out ZIP codes that had only passes or only fails. Using a formula (roughly Bn/B(n-1)), I calculated the fail rate of food inspections in each ZIP, displayed in the third column.


3. Uploading data to Flourish

Taking the ZIP codes and percentage fail rates, I imported this data into Flourish. I was able to make and publish the following horizontal bar chart:



Next, I wanted to make a heat map that could visually represent failed food inspections in Chicago.

1. Filtering the data set

I went back to the same data set. This time, I filtered it so that only "failed" results would show.


2. Uploading the data to Flourish

In a map template in Flourish, I uploaded all the "fail" data points from my spreadsheet, including the establishment name, risk level (for additional detail), and latitude and longitude.


3. Adding ZIP code boundaries

To enhance the visualization, I wanted to add in map lines that would demarcate each ZIP code in the Chicago area. To do this, I downloaded the ZIP Code Boundaries data set from the Chicago Data Portal in its GeoJSON form. I uploaded it directly to Flourish.

For a class project, my group decided to look at the data of national colleges to help prospective college applicants — mostly in the 16–20 range — wrap their heads around their options. I decided to make a map of the top 50 national colleges in the U.S., using data from College Scorecard and U.S. News.

1. Filtering the data set

Using the Most_Recent_Cohorts_Institution dataset from College Scorecard, I filtered the variables to only show what I was interested —  i.e., location (LATITUDE and LONGITUDE), admissions rate (ADM_RATE), tuition (TUITIONFEE_), and whether the school is public or private (CONTROL_PEPS).


2. Filtering the data set by ranking

Then, using the rankings published on U.S. News , I shaved down the data further to only show the sample I'm interested in, or the top 50 schools in the U.S.


3. Uploading the data to Flourish

In a map template in Flourish, I uploaded the data from step 2. I designated rank, acceptance rate, and tuition to appear with the map tooltip. To denote the public/private status more clearly, I used that variable to determine the color of each school's point on the map.
